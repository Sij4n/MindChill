# Mind Chill

Mind Chill is a curated workspace designed to enhance focus, relaxation, and productivity. Combining aesthetic pixel art visuals with lo-fi beats, it provides a cozy virtual environment for study, work, or meditation.

## Features

- **Curated Lofi Streams**: Listen to high-quality lofi beats with interactive backdrops.
- **Pomodoro Timer**: Built-in timer to manage your focus sessions.
- **To-Do List**: Track your tasks and progress.
- **Study With Me**: Peer-to-peer video calling for collaborative study sessions.
- **Customizable**: Minimal, monochrome theme for a distraction-free experience.

## Getting Started

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm start` to launch the application.

## Technologies Used

- React
- React YouTube
- PeerJS (for video calling)
- CSS (Custom Design System)
